# 养猪户猪舍被拆，疑用改装枪射伤3名执法人员后逃走，遭通缉投案自首

极目新闻记者 张万军

福建省南平市延平区茫荡镇，一个养猪户的猪舍被当地拆除后，疑似用改装枪械射伤三名执法人员后逃走。当地警方对其悬赏5万元通缉，当事人目前已投案自首。

![](https://inews.gtimg.com/om_bt/OKTAyzFnGdI3XgioEDX2yt3HBoTYkJbbgtZqJCRi7t3nEAA/1000)_被拆除后的猪舍（受访者提供）_

近日，一份悬赏通缉令在网络上流传，引发网友关注。这份落款显示为南平市公安局延平分局的通缉令显示，2023年5月8日下午，南平市延平区发生一起刑事案件，经查，卢某友有重大作案嫌疑，现在逃。对提供重要有效线索直接抓获犯罪嫌疑人的，警方将予以5万元奖励。

![](https://inews.gtimg.com/om_bt/OuWplokhzUILWfIfbWSnazFk1KaDKFy-
HRbmH5qsonDyUAA/1000)_网上流传的通缉令（来自网络）_

网友称，南平市延平区茫荡镇政府5月8日在宝珠村拆除一个养猪户的猪舍时，当事养猪户用改装霰弹枪射伤执法人员后逃走。

5月16日，南平市延平区茫荡镇宝珠村村民卢先生告诉极目新闻记者，被通缉的卢某友是村里的一名养猪户，其猪舍建在村里的一片荒地上，这块地已经荒废了数十年，猪舍面积约数百平方米。5月8日，茫荡镇政府组织了60多名城管、土地管理、民警等执法人员对卢某友的猪舍进行了拆除。执法人员在离开的路上，遭到卢某友的枪击，卢某友所用的疑似一支改装枪械。此事造成一名民警和两名政府工作人员受伤。事发后，卢某友逃走，4天后投案自首。

卢某友的哥哥卢某向极目新闻记者介绍，弟弟养猪已有几年时间，猪舍所在地是一块荒废了几十年的荒地，以前弟弟在这里种果树，后来在此建了一个几百平方米的猪舍养猪，目前猪舍里养了200多头猪。猪舍拆除前，镇政府的工作人员没有跟弟弟沟通过拆除猪舍的事情。5月8日早晨，镇政府突然组织了几十人和施工机械，赶到弟弟的猪舍，对猪舍进行拆除。弟弟得知消息后，前往猪舍时都被多名工作人员拦在几百米之外。当日10时许，他得知消息后赶到现场时看到，猪舍已被彻底拆除，200多头猪四散逃逸。他在现场和带头的镇政府负责人沟通，要求政府赔偿弟弟的相关损失，但对方没有给出任何回应。

![](https://inews.gtimg.com/om_bt/O1gp_6HAcsmX-pdzJgOKI8i3FG-
OPUtAw81e0OWmUhB5cAA/1000)_事发当日出现在现场的执法人员（受访者提供）_

卢某表示，对于弟弟和镇政府工作人员发生冲突的过程，他不在场，不清楚具体情况。5月12日，弟弟到村委会投案自首，投案之前和自己见了面，让自己帮他把猪卖了帮他还债。

![](https://inews.gtimg.com/om_bt/OuITPK5oFySpdAlYRNqo1fOdOkHXZeN2RAVUs-
Toh8tqQAA/1000)_被拆除后猪舍（受访者提供）_

“开始养猪以来，他先后投入了40多万，大部分钱是我们兄妹和其他人借给他的！这个猪舍是他全部的希望。”卢某表示。

![](https://inews.gtimg.com/om_bt/O6S96rDK65EUQWNrZdaxMA31qmaA-
apdFNGWhcxVqo2M8AA/1000)_被拆除后猪舍（受访者提供）_

茫荡镇政府工作人员告诉极目新闻记者，此事中有三名人员受伤，伤者现在伤势稳定，没有生命危险，嫌疑人已经自首。

极目新闻记者致电南平市公安局欲了解此事，一名工作人员表示领导不在。

南平市委相关部门工作人员向极目新闻记者介绍，当地相关部门的行动是依法执法，程序没有问题，关于此事的详细情况他们将发布通报。
