# 给救护车让行会扣分？交警部门：礼让救护车不会扣分，拒不相让会被罚

**人民日报健康客户端 侯佳欣**

近日，四川成都一红绿灯路口，一救护车被私家车挡住去路，私家车司机因担心被拍照扣分拒绝让行。视频中，该司机称“拍照扣分怎么办”，工作人员表示，如被拍，可帮忙取消，但司机仍然拒绝。

5月28日，人民日报健康客户端记者致电北京、上海、成都等多地交警部门，工作人员均表示，如果是给执行紧急任务的救护车让道，而出现扣分等处罚，只要具备真实有效的证据，可以向交管部门提出申请，相关部门查看监控确认属实后，会给让行的车辆消除违法信息，并撤销处罚。

5月27日，据九派新闻，双流区交警大队一工作人员表示，此事正在调查处理。该工作人员称，礼让救护车不会扣分，如误扣可申诉取消，拒不相让的交警有权对其扣分处罚。

根据《中华人民共和国道路交通安全法》第五十三条规定，警车、消防车、救护车、工程救险车执行紧急任务时，可以使用警报器、标志灯具；在确保安全的前提下，不受行驶路线、行驶方向、行驶速度和信号灯的限制，其他车辆和行人应当让行。

此外，根据《道路交通安全违法行为处理程序规定》第二十二条第三款规定，交通技术监控设备记录或者录入道路交通违法信息管理系统的违法行为信息，有下列情形之一并经核实的，违法行为发生地或者机动车登记地公安机关交通管理部门应当自核实之日起三日内予以消除，其中明确的第一种情形就是“警车、消防救援车辆、救护车、工程救险车执行紧急任务期间交通技术监控设备记录的违法行为”。

“如果是因为让道正在执行任务的120而闯了红灯，这种情况不会被扣分。如果误扣，可直接在12123
小程序进行申诉，如线上申诉不通过或遇到什么问题，还可到事发辖区交警队进行解决。”四川省成都市交警一分局工作人员向人民日报健康客户端透露，一般来说，在收到线上申诉反馈后，相关交警部门会查看事故监控，确认情况属实后，就会为让行车辆消除违法信息并撤销处罚。

相反，如果遇到正在执行任务的救护车不让行，还有可能被处罚。根据《中华人民共和国治安管理处罚法》第五十条规定，阻碍执行紧急任务的消防车、救护车、工程抢险车、警车等车辆通行的，处警告或者200元以下罚款；情节严重的，处5日以上10日以下拘留，可以并处500元以下罚款。
