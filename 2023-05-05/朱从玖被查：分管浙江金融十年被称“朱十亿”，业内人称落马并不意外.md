# 朱从玖被查：分管浙江金融十年被称“朱十亿”，业内人称落马并不意外

5月4日下午，中央纪委国家监委宣布，浙江省政协党组成员、副主席朱从玖涉嫌严重违纪违法，目前正接受中央纪委国家监委纪律审查和监察调查。

今年58岁的朱从玖，作为副省长曾分管浙江金融十年之久。在赴任浙江之前，他在金融业已有二十三年工作经历，其中前四年在央行，后十九年在证监会和上交所。2012年5月调任浙江省副省长前，朱从玖已是证监会主席助理、党委委员。

对于朱从玖的被查，有业内人士表示“并不意外”，因为近期已有多名与朱从玖有过工作交集的金融界人士被查。

**上交所最年轻的总经理**

公开资料显示，朱从玖，1965年生，安徽省肥西县人。1981年9月，16岁的他考入安徽财贸学院财政金融系财政金融专业学习，本科毕业后又进入俗称为“五道口”的人民银行研究生部继续深造。1988年研究生毕业后，朱从玖进入中国人民银行，在金融管理司工作了四年。

1986年至1992年这段时间，我国的证券主管部门是中国人民银行。1992年10月，国务院证券委及其监管执行机构中国证监会宣布成立。朱从玖同年进入证监会办公室工作。1999年，朱从玖赴上海证券交易所担任副总经理。第二年，年仅35岁的朱从玖就成为上交所历史上最年轻的总经理。2008年，朱从玖重返证监会，担任主席助理兼发行部主任，直至2012年调任浙江。

作为专家型金融监管干部，朱从玖参与了中国证券市场的从无到有、从小到大的过程，历经了股票发行从审批制到核准制的过渡，再到注册制的酝酿。

在上交所的8年时间，朱从玖一直致力于金融创新，包括推出上证50指数、推动开发指数ETF，推动四大国有银行、中国石油、宝钢集团、长江电力、中国神华等巨型蓝筹企业在A股的上市以及推出了国债回购。

**空降浙江操刀温州金改**

2012年，47岁的朱从玖从证监会主席助理位置上“空降”浙江担任副省长，成为当时还不算多见的金融副省长，也是最年轻的金融副省长。2018年10月，全国已有12名金融副省长，其中6名出自国有大行，6名来自央行和证监会，只有朱从玖一人具有央行和证监会的双重工作经历。

朱从玖履新浙江的两个月前，温州刚刚经历了民间借贷风波，获批成为全国首个金融综合改革试验区。作为金融老将，外界对朱从玖寄予厚望，人们希望他能迅速整治当时浙江民间借贷爆雷的乱象。

他上任没几天，便赴温州调研，履新后首次在媒体面前公开亮相，就是畅谈温州金改。到任半年后，“温州金改12条细则”正式出台，温州金改轰轰烈烈搞了起来。2013年4月，他还远赴北京推动《浙江省温州民间融资管理条例》的立法，七个月后该条例正式出炉，成为我国第一部民间借贷的地方性法规，从制度上实现了民间借贷规范化、阳光化、法制化。

在民营经济发达、小微企业众多的浙江，朱从玖把推进普惠金融作为浙江深化金融供给侧结构性改革、实现金融高质量发展的重要主线。2013年，浙江在全国率先实施“普惠金融工程三年行动计划（2013—2015年）”，提出要打通金融服务“最后一公里”。

2019年11月，温州成为深化民营和小微企业金融服务综合改革首批试点城市。而台州则打造了小微金融“台州模式”，双州一直成为浙江的两面金改旗帜。

作为证监会出身的专家型干部，朱从玖比其他官员更深谙直接融资对于民企发展的重要意义。在他的推动下，浙江省在2017年10月提出了旨在推进企业上市和并购重组的“凤凰行动”计划，并在2019年推出了2.0版。截至2020年11月，浙江新增了238家上市公司，主板、中小板、创业板和科创板的上市公司数量分别名列全国第1、第2、第4和第5位。

当然，朱从玖在浙江主管金融的这十年，也遇到过重大挑战。2018年夏天，包括浙江在内的全国多地都爆发了又一轮P2P爆雷潮。当时有统计显示，2018年6到7月，爆雷潮波及全国264家P2P平台，其中81家平台位于浙江，仅省会杭州就有69家平台爆雷，这一数字接近当时全国总数量的四分之一，杭州也因此被人成为“雷都”。

有人认为，浙江当时刹车最晚，后果最严重，主管金融的副省长朱从玖有失职之嫌。

**申报家财被称“朱十亿”**

年轻有为的朱从玖在副省长的位置上干了10年，成为任职时间最长的金融副省长，却始终未能在仕途上再进一步。去年7月，朱从玖不再担任副省长，成为浙江省政协党组成员，今年1月当选浙江省政协副主席。

据财新报道，朱从玖没有“更进一步”主要是因为家里太有钱了。据知情人士透露，朱从玖多年以来就是证监会及浙江省高级干部里公开的“有钱人”，“向组织申报的财产就有几个亿”，他在当地坊间甚至有“朱十亿”的戏称。

据报道，朱从玖在上交所任职期间，工资很高，当时上海房价低，朱从玖在那个时间段投资了多套上海房产。上海的别墅2000年初价值几十、几百万，后来市值过亿，也属于平常。此外，朱从玖的妻子从事收入较高的私募基金行业。

有业内人士指出，朱从玖作为体制内官员却有如此丰厚的家产，且其妻的职业与其分管的领域密切有关，确实很容易让外界产生微妙的联想。

时至今日，朱从玖的家庭财产是否合规合法，还需要等待权威部门的确认。

**多位“五道口”同窗被查**

朱从玖究竟因何被查，尚无权威部门披露详情。值得一提的是，近年来，朱从玖的多名同学和同事已先期被查，有人甚至已获刑入狱。

公开报道显示，1985级硕士班是五道口开班以来扩招的首届，也成为历届毕业生中“出事”最多的一届。与朱从玖同在1985年考入五道口的硕士同窗中，已有多位落马。比如3月底落马的中国银行原董事长刘连舸，2015年9月落马的证监会原主席助理张育军，2019年8月自首、2022年获刑的证大投资集团董事长戴志康。此外，今年2月被免职而提前退休的中国人保集团原董事长罗熹也是朱从玖的同班同学。

据报道，这其中，张育军与朱从玖的人生轨迹重合最多。两人同年大学毕业，同年就读于“五道口”，毕业后都在央行金融管理司工作数年，后来又都进了证券监管部门。两人先后离开证监会，一个去了上交所，一个去了深交所。朱从玖从上交所总经理上调证监会主席助理，又从主席助理调任浙江，两次补他“空缺”的都是张育军。

2015年，张育军被查。2017年7月，中央纪委通报了对张育军严重违纪问题进行立案审查的结果。通报称，张育军严重违反政治纪律和组织纪律，对抗组织审查，不按规定报告个人有关事项；违反中央八项规定精神，违规打高尔夫球，接受可能影响公正执行公务的宴请；违反廉洁纪律和工作纪律，纵容、默许亲属利用其职务影响谋取巨额不当利益；违反生活纪律。其中，部分问题涉嫌犯罪。

通报指出，张育军身为党的高级领导干部，理想信念丧失，党性观念和纪律意识淡漠，“靠山吃山”，扰乱资本市场秩序，损害证券监管部门形象，严重违反党的纪律，性质恶劣、情节严重。依据相关规定，决定给予其开除党籍和公职的处分；将其涉嫌犯罪线索及所涉款物移送司法机关依法处理。

2017年9月，最高人民检察院经审查决定，依法对张育军以涉嫌受贿罪立案侦查并采取强制措施。据财新网2021年9月1日报道，仍被调查中的张育军因胰腺癌病逝，时年58岁。

**多名证监会时期下属被查**

除张育军外，近期证监会及交易所多名“落马”人员，也与朱从玖有工作交集。4月21日，上交所原副总经理刘逖接受调查，此前还有证监会会计部原主任王宗成，以及证监会发审委委员、上交所原科创板上市审核中心副主任操舰被查。王宗成曾于2016年7月至2018年12月任证监会浙江监管局局长，2022年6月被查。操舰于2009年9月至2019年1月在上交所公司管理部、发行上市部、发行上市中心任职，其中2010年4月至2012年5月、2014年5月至2017年9月，任多届主板发审委委员。

在浙江，朱从玖此次被查，最先被人联想到的是浙商证券总裁王某。

有报道称，3月31日，王某在参加浙江省交通投资集团财务有限公司干部会议时，被“叫出会场”。随后公司回应，称总裁在“休假”。不过，4月18日，有媒体报道，据接近浙商证券的人士透露，王某已回到公司，“明日将出席公司党风廉政会议。”

不过，北青报记者从浙商证券官网了解到，4月20日，浙商证券召开了2023年党风廉政建设大会暨一季度党建纪检工作会议；同日，浙商证券改革发展动员大会在浙江诸暨开幕。这两场会议的报道并未提到总裁王某的名字。

据报道，今年42岁的王某毕业于北大法学院，毕业后进入证监会发行部工作，2008年开始担任分管发行的主席助理朱从玖的秘书。2012年，朱从玖调任浙江省副省长，王某也随之前往浙江。

浙商证券2022年年报显示，王某曾任浙江省交通投资集团财务有限责任公司副总经理、总经理。2016年1月至2017年7月任浙商证券监事会主席，2017年8月至今任浙商证券董事、总裁。
